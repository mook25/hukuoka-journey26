Visit Japan Web (비짓재팬웹)

  • 일본 입국심사 + 세관신고를 온라인으로 미리 처리하는 서비스
  • 출발 전에 반드시 3명 모두 등록 (대표 1명이 동반가족 등록 가능)
  • 등록 후 발급되는 QR코드를 일본 도착 시 입국심사대와 세관에서 보여주면 됨
  • 등록 사이트: https://services.digital.go.jp/en/visit-japan-web/
  • 등록 절차: 계정 생성 → 여행일정 등록 → 여권정보 입력 → 입국심사 정보 입력 → 세관신고 입력 → QR코드 발급
  • 스마트폰에 QR코드를 저장하거나, PC에서 인쇄해 가세요

체크인 방법 (단계별)

1 티웨이항공 온라인 체크인 페이지 접속

  • 아래 링크를 클릭하거나, 티웨이항공 앱에서 "온라인 체크인" 메뉴를 선택하세요
  • 👉 티웨이 온라인 체크인 바로가기

2 예약번호 + 탑승객 정보 입력

  • 예약번호: BALPTX
  • 영문 성(Last Name): 대표 탑승객의 영문 성 입력 (예: KIM)

3 여행편 선택

  • 체크인할 항공편을 선택합니다 (가는편 / 오는편)
  • 3명 모두 한 번에 체크인할 수 있습니다

4 여권 정보 입력 (국제선 필수)

  • 탑승객별로 여권번호, 여권 만료일, 국적, 생년월일을 입력합니다
  • 여권을 보면서 정확히 입력해 주세요

5 약관 동의

  • 3개 항목 모두 동의 체크 후 다음으로 진행

6 좌석 선택

  • 좌석 배치도에서 원하는 좌석을 선택합니다
  • 3명이 나란히 앉으려면 같은 줄의 연속 좌석을 선택하세요

7 체크인 완료 → QR 탑승권 저장

  • 체크인이 완료되면 QR코드가 포함된 모바일 탑승권이 발급됩니다
  • 반드시 스마트폰에 캡처/저장하거나, 종이로 인쇄해 주세요!
체크인 완료 후, 공항에서는?

부치는 짐(수하물)이 있는 경우:
QR 탑승권 저장 → 공항 도착 → 수하물 카운터에서 짐 위탁 → 출국심사 → 탑승구

부치는 짐이 없는 경우:
QR 탑승권 저장 → 공항 도착 → 바로 출국심사 → 탑승구

※ 보안검색대와 탑승구에서 QR 탑승권을 보여주시면 됩니다.
※ 온라인 체크인을 하면 공항 체크인 카운터에서 줄을 서지 않아도 됩니다!
