## 1. 출발 전 준비사항

### 비자
- 한국 국민은 90일 이내 관광 목적 방문 시 **비자 면제**. 별도 신청 불필요.

### Visit Japan Web (비짓재팬웹) - 필수 등록 권장
- 일본 입국심사 + 세관신고를 온라인으로 미리 처리하는 서비스
- **출발 전에 반드시 3명 모두 등록** (대표 1명이 동반가족 등록 가능)
- 등록 후 발급되는 **QR코드**를 일본 도착 시 입국심사대와 세관에서 보여주면 됨
- 등록 사이트: https://services.digital.go.jp/en/visit-japan-web/
- **등록 절차**: 계정 생성 → 여행일정 등록 → 여권정보 입력 → 입국심사 정보 입력 → 세관신고 입력 → QR코드 발급
- 스마트폰에 QR코드를 저장하거나, PC에서 인쇄해 가세요

### 기타 준비물
- **여권** (유효기간 확인 필수)
- **항공권 예약 확인서** (스마트폰 캡처 또는 인쇄)
- **숙소 예약 확인서** 모두 인쇄 (스기노이호텔, 더블러썸 하카타)
- **고속버스 WEB승차권** 2장 (가는편/오는편) - 스마트폰 화면 또는 인쇄본 필수 (예약 완료 메일로는 탑승 불가!)
- **해외 사용 가능 신용/체크카드**
- **여행자보험** 가입 권장
- **일본 유심/eSIM 또는 포켓와이파이** (네비게이션, 번역앱 사용 위해 필수)

## 3. 후쿠오카 공항 도착 후 → 벳푸행 버스 탑승

비행기는 **11:30 도착**, 버스는 **12:42 출발**이므로 약 1시간 12분의 여유가 있습니다.

### 단계별 안내

1. **비행기 하차 → 입국심사**
   - Visit Japan Web QR코드를 입국심사대에서 제시
   - 외국인 전용 카운터로 이동 (지문 촬영 + 사진 촬영)

2. **수하물 수취**
   - 짐 찾는 곳(Baggage Claim)에서 짐을 찾습니다

3. **세관 통과**
   - Visit Japan Web 세관 QR코드 제시하면 빠르게 통과

4. **버스 정류장으로 이동**
   - 국제선 도착 1층 밖으로 나오면 바로 앞에 버스 정류장이 있습니다
   - **2번 승강장**에서 벳푸행 고속버스 탑승
   - 승차 시 **WEB승차권 QR코드 화면** (또는 인쇄본)을 운전기사에게 보여주세요
   - 좌석: **04A(남), 04B(여), 04C(남)**

> 입국에서 세관까지 보통 30-40분 소요됩니다. 시간 여유가 있으니 당황하지 마세요.
